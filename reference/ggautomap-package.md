# Create Maps From a Column of Place Names

Mapping geometries based on `{ggplot2}` for turning a spreadsheet into
maps. This package provides ggplot geoms that make maps from a data
frame where one of the columns contains place names. The map data must
be registered with `{cartographer}`.
